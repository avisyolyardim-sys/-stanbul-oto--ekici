# İstanbul Oto Kurtarıcı Web Sitesi

Bu proje GitHub Pages üzerinde yayınlanmaya hazır statik web sitesidir.

## GitHub Pages ile yayınlama

1. GitHub üzerinde yeni bir repository oluşturun.
2. Bu klasördeki tüm dosya ve klasörleri repository kök dizinine yükleyin:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `assets/`
   - `bolgeler/`
   - `hizmetler/`
   - `sitemap.xml`
   - `robots.txt`
   - `.nojekyll`
3. GitHub'da `Settings > Pages` bölümüne girin.
4. `Build and deployment` kısmında `Deploy from a branch` seçin.
5. Branch olarak `main`, klasör olarak `/root` seçin.
6. Kaydettikten sonra site birkaç dakika içinde yayınlanır.

## SEO notu

Şu an canonical ve sitemap adresleri `https://www.yolyani-cekici.com/` domainine göre hazırlanmıştır.
GitHub Pages URL'si ile yayınlanacaksa bu adresler gerçek GitHub Pages adresiyle değiştirilmelidir.
Örneğin:

```text
https://kullaniciadi.github.io/repo-adi/
```

Gerçek domain bağlanacaksa GitHub Pages ayarlarından custom domain eklenmelidir.
